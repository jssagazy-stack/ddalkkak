빠른 시작 가이드

처음 사용하시나요? 아래 5단계만 따라하세요.

  1. 프로그램 실행 → 관리자 권한 승인
  2. 신규등록 / 로그인
  3. 대상 창 선택
  4. 좌표 설정 & 키 매핑
  5. ON 켜고 키 눌러보기

시작하기

STEP 1. 프로그램 실행

  1. 딸깍.exe를 더블클릭하여 실행합니다.
  2. 관리자 권한 승인 창이 뜨면 예를 클릭합니다.
관리자 권한이 없으면 프로그램이 정상 작동하지 않습니다.

STEP 2. 신규등록 / 로그인

처음 사용하는 경우 (신규등록)

  1. 신규등록 탭을 선택합니다.
  2. 아이디와 비밀번호를 입력합니다.
  3. 신규등록 버튼을 클릭합니다.
  4. 가입 완료 후 로그인 화면으로 돌아갑니다.

신규등록 화면

기존 회원 (로그인)

  1. 아이디와 비밀번호를 입력합니다.
  2. 로그인 버튼을 클릭합니다.

로그인 화면

STEP 3. 대상 창 선택

로그인이 완료되면 대상 창 선택 안내가 나타납니다.

  1. 안내 메시지에서 확인을 클릭합니다.
  2. 사용할 프로그램 창 위에 마우스를 올리고 클릭합니다.
  3. 선택 완료 메시지가 표시되면 성공입니다.

대상 창 선택 - 사용할 프로그램 창을 클릭하세요

반드시 사용할 프로그램 창을 클릭하세요.
바탕화면이나 다른 프로그램을 클릭하면 정상 작동하지 않습니다.

기본 설정

대상 창을 선택했다면, 좌표 설정과 키 매핑을 해야 합니다.

ON/OFF 버튼 이동하기

화면에 표시되는 ON / OFF 버튼은 원하는 위치로 옮길 수 있습니다.

  1. Ctrl을 누른 채로 버튼을 우클릭 드래그합니다.
  2. 원하는 위치에서 마우스를 놓으면 버튼이 이동합니다.
버튼이 다른 창에 가려질 때는, 시스템 트레이(화면 오른쪽 하단)의 딸깍 아이콘을 우클릭하면 동일한 메뉴를 사용할 수 있습니다.

설정 메뉴 열기

아래 방법 중 하나로 설정 메뉴를 엽니다.

  • 화면의 ON / OFF 버튼을 우클릭
  • 화면 오른쪽 하단 시스템 트레이에서 딸깍 아이콘을 우클릭

우클릭 메뉴 - 모든 설정에 접근할 수 있습니다

STEP 4-1. 좌표 설정

목적: 단축키(F5~F12)가 클릭할 화면 위치를 지정합니다.

  1. 우클릭 메뉴에서 좌표 설정을 클릭합니다.
  2. 설정할 단축키 버튼 (예: F5 좌표)을 클릭합니다.
  3. 안내 메시지가 나타나면, 대상 창에서 원하는 위치를 우클릭합니다.
  4. 좌표가 저장됩니다. (예: F5: (645, 873))
  5. 필요한 다른 키도 같은 방법으로 설정합니다.
  6. 확인을 클릭하여 저장합니다.

좌표 설정 - 각 단축키가 클릭할 위치를 우클릭으로 지정

STEP 4-2. 키 매핑

목적: F5~F12 키를 누르기 편한 키(Q W E R 등)로 변경합니다.

  1. 우클릭 메뉴에서 키 매핑 설정을 클릭합니다.
  2. 변경할 키 버튼을 클릭합니다. (예: F5 옆의 Q 버튼)
  3. 원하는 키를 키보드에서 누릅니다. (ESC = 취소)
  4. 휠 클릭을 사용하려면 해당 키 옆의 체크박스를 체크합니다.
  5. 확인을 클릭하여 저장합니다.

키 매핑 설정 - F5~F12를 원하는 키로 변경

휠 클릭이란?
체크하면 해당 키를 누를 때 지정된 좌표에서 마우스 휠 버튼(가운데 버튼)을 클릭합니다.
체크하지 않으면 일반 좌클릭으로 동작합니다.

ON/OFF 전환 키 변경 (선택)

기본 전환 키는 ` (숫자 1 왼쪽 키)입니다. 원하는 키로 변경할 수 있습니다.

  1. 키 매핑 설정에서 ON/OFF 키 버튼을 클릭합니다.
  2. 원하는 키를 누릅니다.

프리셋 (프로 전용)

키 매핑 설정을 A, B, C 3개 프로파일로 저장할 수 있습니다.

예: A = 기사 설정, B = 요정 설정, C = 법사 설정

타이밍 설정은 기본값 그대로 사용해도 충분합니다.
고급 사용자만 필요 시 조정하세요. (우클릭 메뉴 → 통합 설정)

사용 방법

STEP 5. ON / OFF 전환

설정이 끝났으면 기능을 켜고 사용합니다.

켜는 방법 (2가지)

  • 대상 창이 활성화된 상태에서 ` 키를 누릅니다.
  • 화면의 OFF 버튼을 직접 클릭합니다.

끄는 방법

  • 같은 키(`)를 다시 누릅니다.
  • ON 버튼을 클릭합니다.
  • 대상 창에서 다른 창으로 전환하면 OFF 됩니다.

ON/OFF 버튼 - 클릭 또는 단축키로 전환

안전 기능: 대상 창이 비활성화되면 OFF로 전환됩니다.
다른 프로그램에서 키가 오작동하는 것을 방지합니다.

매핑된 키 사용하기

예시: F5를 Q로 매핑, 휠 클릭 체크한 경우

  1. ON 상태로 전환합니다.
  2. 대상 창에서 Q 키를 누릅니다.
  3. F5에 설정한 좌표에서 휠 클릭이 실행됩니다.
마우스 커서 위치는 변하지 않습니다. 현재 위치 그대로 지정된 좌표를 클릭합니다.

반복 실행 (프로 전용)

키를 계속 반복해서 실행할 수 있습니다.

동작 방법
반복 시작 Shift + 매핑된 키 (예: Shift + Q)
반복 중지 해당 키만 누름 (예: Q) 또는 OFF 전환
여러 키를 동시에 반복할 수 있습니다. 각 키의 반복은 개별적으로 시작/중지됩니다.

추가 기능

오버레이 버튼

화면에 클릭 가능한 버튼을 띄워서 사용할 수 있습니다.

버튼 표시/숨기기

  • 우클릭 메뉴 → 버튼 보이기/숨기기

오버레이 버튼 - 화면에 띄워서 클릭

버튼 사용 방법

  • 좌클릭 → 지정된 동작 실행
  • Shift + 좌클릭 → 반복 실행 (프로 전용)
  • 우클릭 → 버튼 설정 메뉴
  • Ctrl + 우클릭 드래그 → 버튼 위치 이동

버튼 설정

우클릭 메뉴 → 버튼 UI 설정에서 아래 항목을 조정할 수 있습니다.

  • 버튼 개수, 행 수, 방향 (가로/세로)
  • 버튼 크기 (너비, 높이), 간격
  • 투명도

버튼 UI 설정 - 개수, 크기, 배치 등 조정

시퀀스

여러 동작을 순서대로 묶어서 한 번에 실행할 수 있습니다.

예시: "F5 클릭 → 0.5초 대기 → F6 클릭" 같은 연속 동작

시퀀스 만들기

  1. 우클릭 메뉴 → 시퀀스 설정을 클릭합니다.
  2. 새 시퀀스를 클릭하고 이름을 입력합니다.
  3. 동작(액션)을 추가합니다: 클릭, 키 입력, 딜레이 등
  4. 확인으로 저장합니다.

시퀀스 설정 - 여러 동작을 순서대로 구성

시퀀스를 버튼에 연결하기

  1. 오버레이 버튼을 우클릭합니다.
  2. 시퀀스 지정을 선택합니다.
  3. 목록에서 시퀀스를 선택합니다.
  4. 이제 해당 버튼을 클릭하면 시퀀스가 실행됩니다.
시퀀스는 단축키에도 연결할 수 있습니다. 키 매핑 설정에서 원하는 키에 시퀀스를 지정하세요.

등급 안내

딸깍은 베이직(무료)과 프로(후원제) 두 가지 등급이 있습니다.

기능 베이직 (무료) 프로 (후원제)
키 매핑 슬롯 10개 50개
프리셋 A만 A ~ C
오버레이 버튼 1개 30개
시퀀스 1개 50개
반복 실행 - O
휠 클릭 - O

프로 등급 전환

  1. 우클릭 메뉴 → 통합 설정 → 내 정보 탭
  2. 연장 신청 버튼을 클릭합니다.
  3. 입금 후 관리자 승인을 기다립니다.
  4. 승인 완료 후 프로그램을 재시작하면 프로 등급이 적용됩니다.
프로 만료 시 주의: 베이직으로 전환되면 일부 설정이 삭제될 수 있습니다.
만료 전에 설정 내보내기로 백업을 권장합니다.

문제 해결

프로그램이 실행되지 않아요

  • 관리자 권한으로 실행했는지 확인하세요. (UAC 창에서 "예" 클릭)
  • 인터넷 연결 상태를 확인하세요. (온라인 인증 필요)

키를 눌러도 반응이 없어요

  • ON 상태인지 확인하세요.
  • 대상 창이 활성화되어 있는지 확인하세요.
  • 좌표 설정을 했는지 확인하세요.
  • 키 매핑을 했는지 확인하세요.

해결: 우클릭 메뉴 → 대상 창 선택으로 다시 지정해보세요.

다른 프로그램에서도 키가 눌려요

  • 대상 창 선택이 잘못되었을 수 있습니다.
  • 우클릭 메뉴 → 대상 창 선택에서 정확히 사용할 창을 다시 클릭하세요.

휠 클릭이 안 돼요

  • F5~F12 키만 휠 클릭이 가능합니다.
  • 키 매핑 설정에서 해당 키의 휠 클릭 체크박스가 체크되어 있는지 확인하세요.
  • 프로 등급에서만 사용 가능합니다.

오버레이 버튼이 안 보여요

  • 우클릭 메뉴 → 버튼 보이기/숨기기를 클릭하세요.
  • 시스템 트레이에서 딸깍 아이콘이 있는지 확인하세요.

시퀀스가 실행되지 않아요

  • ON 상태인지 확인하세요.
  • 대상 창이 활성화되어 있는지 확인하세요.
  • 시퀀스에 동작(액션)이 추가되어 있는지 확인하세요.
  • 버튼에 시퀀스가 지정되어 있는지 확인하세요.

라이선스가 만료되었어요

  • 프로그램 실행 시 만료 안내가 표시됩니다.
  • 연장 신청을 클릭하고 안내에 따라 진행하세요.
  • 승인 후 프로그램을 재시작하면 됩니다.
